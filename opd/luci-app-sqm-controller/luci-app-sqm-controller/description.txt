SQM Controller 是一个基于 OpenWrt 的智能队列管理（Smart Queue Management, SQM）控制器，
提供图形化 LuCI Web 界面，用于配置和管理流量整形与队列调度算法。

本应用支持 fq_codel 与 cake 两种常用队列算法，可用于缓解网络缓冲膨胀（Bufferbloat），
提高多设备、多连接环境下的网络响应速度和公平性。

主要特性：
- 支持 fq_codel / cake 算法自由选择
- 支持上行与下行带宽独立设置
- 使用 tc + ifb 实现下载方向整形
- 提供服务启动 / 停止 / 重启控制
- 提供实时状态监控页面（tc 规则与运行状态）
- 提供系统日志查看、自动刷新与高亮错误显示
- 支持开机自启动与配置持久化（UCI）

适用平台：
- OpenWrt 23.05 及以上版本
- 需要内核支持：ifb、htb、fq_codel、cake

本项目定位为轻量级 SQM 控制工具，不依赖官方 sqm-scripts，
适合自定义部署与二次开发。
